# HUGO compiler

## постулат ExeBoi (Акт I)
Это проект который должен реализовать минимальный компилятор c23. На момент 26.06.2026 есть только tcc, который создаёт просто минимльный компилятор c. О функционале c26 ему только мечтать. Так же нигде не было компилятора который поддерживает мульти платформиную компиляцию (приходится компилировать самому). Хочется что бы этот проект стал эталоном компиляторов в последующем.
